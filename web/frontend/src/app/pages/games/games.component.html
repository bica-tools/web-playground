The Lattice Is the Board

Session types and game theory share deep structural connections.

🎯

Two Players

Client controls external choice (&{...}). Server controls internal choice (+{...}). Every protocol is a two-player game.

🏁

Strategies Are Paths

A strategy is a path from ⊤ to ⊥ through the Hasse diagram. The lattice guarantees every game terminates with a well-defined outcome.

⚖

Meet & Join

Meet is where two execution paths converge. Join is where they split. The lattice structure guarantees these always exist.

🔄

Recursion = Loops

Recursive types create cycles in the game tree. The game can loop — but termination analysis guarantees it will eventually reach ⊥.
